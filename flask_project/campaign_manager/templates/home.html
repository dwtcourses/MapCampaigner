Manage mapping activities

Map Campaigner is a tool that is intended to gather all project managers and also surveyors into one platform.

Project manager can arrange mapping activities with specific features and attributes according to their needs. Map Campaigner will monitor the progress of the activity in a statistical data about how many objects have been mapped as well as how many data collectors are engaged.
